AngularJS Code Camp: Building Web App with AngularJS

December 13, 2014 9AM - 5PM
Divine Word College of Legazpi, Rizal St Corner Fr. Bates St
Old Albay, Legazpi City

Register Now!

Interested in diving into AngularJS and learning how it can be used to build robust Single Page Applications? The code camp is hands-on with 3 hours lecture of basic foundation of AngularJS. Participants will participate in a mini-hackathon based on what they have learned during the overview session.

This is the first code camp in our series in preparation for #BicolITHackathon2015. A ‘hackathon’ is an event wherein participants collaborate to build a viable product that can spark a future business.gives freebies to the attendees. Come and join us as we unfold the new trends of Technology.


Speaker

John Kevin Marbella Basco CTO, Mayon Volcano Software

John is also the co-founder of Real Problem Hunt, the Product Hunt for problems, and the creator of Laravel Snippets, a repository of code snippets for the Laravel PHP Framework.

Prerequisites

Participants should have fundamental knowledge with HTML, JavaScript, and jQuery or another DOM manipulation library.

For those who will bring their laptops, make sure the following are installed:

  • Chrome (web browser). We will use the Chrome debugging tools, Chrome extension for AngularJS (ng-inspector) and a Chrome extension called Post Man.
  • Text editor or JavaScript-capable IDE.

Slots are limited, kindly notify us in advance if you can't make it on that date.

We conduct regular technology talks for free for students and professionals to augment the training needs and expose others to the latest trends in information technology.
